# 📅 Custom Range Calendar (SwiftUI)

A fully custom SwiftUI calendar with smooth paging and date range selection.

## ✨ Features

* Select single date or date range
* Horizontal paging between months
* Prevents selecting future dates
* Animated selection & range highlighting
* Custom day cells with dynamic states (today, selected, in-range)
* Built-in save action for selected range

## 🧩 Architecture

* `CustomCalendarView` → main calendar logic
* `MonthView` → month grid rendering
* `DayCell` → individual day UI & states
* `CalendarConstants` → design & behavior config

## 🚀 Tech

SwiftUI • LazyVGrid • TabView (paging) • Animations • Date handling

A clean, flexible, and production-ready calendar component.
